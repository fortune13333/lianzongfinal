# “链踪”V5.0 功能说明：动态配置模板

## 1. 功能概述：从“静态”到“动态”的革命

欢迎来到“链踪”V5.0！本次更新我们引入了一项革命性的功能——**动态配置模板 (Dynamic Configuration Templates)**。

在之前的版本中，模板是“静态”的。如果您想为10台设备设置不同的主机名或IP地址，您可能需要创建10个相似的模板，或者在部署前手动修改模板内容。这不仅效率低下，而且极易出错。

**动态配置模板彻底解决了这个问题。** 它允许您在模板中嵌入“占位符”或“变量”。在执行批量部署时，“链踪”的后端代理会自动将这些变量替换为每台设备的具体信息，从而实现用**一个模板**，为**多台设备**生成**独一无二的、个性化的配置**。

这项功能是“链踪”从一个配置记录工具，迈向一个真正的**网络自动化平台**的关键一步。

---

## 2. 核心原理：智能变量替换

动态模板的核心原理非常简单直接：**字符串替换**。

我们在后端代理的批量部署逻辑 (`/api/bulk-deploy`) 中增加了一个智能处理层。当代理接收到部署任务时，它会为每一台目标设备执行以下操作：

1.  **加载模板**: 从数据库 (`chaintrace_data.json`) 中读取您选择的模板内容。
2.  **获取设备信息**: 读取当前目标设备的详细信息（ID, 名称, IP地址）。
3.  **执行替换**: 在模板内容中查找所有预定义的变量占位符，并用设备的实际信息进行替换。
4.  **生成最终配置**: 生成一份为该设备“量身定制”的完整配置。
5.  **创建新区块**: 将这份最终配置，连同其他元数据，作为一个新的区块记录到该设备的区块链历史中。

### 可用的动态变量

目前，您可以在模板中使用以下三个核心变量：

| 变量占位符 | 自动替换为 | 示例 |
| :--- | :--- | :--- |
| `{{ device.id }}` | 设备的唯一ID | `RTR01-NYC` |
| `{{ device.name }}` | 设备的友好名称 | `Core Router NYC` |
| `{{ device.ipAddress }}` | 设备的IP地址 | `192.168.1.1` |

## 3. 使用方法：两步实现高效部署

### 第一步：创建或编辑一个动态模板

1.  以 `admin` 用户登录，进入**管理中心**。
2.  切换到 **“配置模板”** 选项卡。
3.  点击“添加新模板”或编辑一个现有模板。
4.  在“配置内容”文本框中，像编写普通配置一样编写您的模板，并在需要个性化的地方，插入上述的动态变量。

**示例：创建一个标准的交换机接口配置模板**

假设我们希望所有接入层交换机的 `GigabitEthernet0/1` 端口都启用 `port-security`，并且接口描述中要包含设备自己的名字，以便于快速识别。

您的模板内容可以这样写：

```ini
interface GigabitEthernet0/1
 description -- Access Port for {{ device.name }} --
 switchport mode access
 switchport port-security
 switchport port-security maximum 2
 switchport port-security violation restrict
 switchport port-security mac-address sticky
 spanning-tree portfast
!
```

### 第二步：执行批量部署

1.  返回**仪表盘**。
2.  在您想要部署的设备卡片上，勾选左上角的**复选框**，选择多台目标设备。
3.  底部的“批量操作工具栏”将会出现。点击 **“部署模板”** 按钮。
4.  在弹出的“批量部署”窗口中：
    *   从下拉菜单里选择您刚刚创建的那个动态模板。
    *   您会看到模板内容的预览，其中仍然显示着 `{{ device.name }}` 等变量。
    *   下方会列出所有您已选择的目标设备。
5.  点击 **“确认部署”**。

**完成！** “链踪”的后端代理现在会自动为每一台选中的交换机生成并记录一份独特的配置。例如，对于名为 `Access-SW-Floor10` 的设备，它记录的新配置将会是：

```ini
interface GigabitEthernet0/1
 description -- Access Port for Access-SW-Floor10 --
 switchport mode access
 switchport port-security
 switchport port-security maximum 2
 switchport port-security violation restrict
 switchport port-security mac-address sticky
 spanning-tree portfast
!
```

## 4. 核心价值与毕业设计亮点

在您的毕业设计论文或答辩中，这项功能是一个极佳的亮点，因为它充分体现了：

*   **自动化思维**: 展示了您不仅仅满足于手动操作，而是主动思考如何将重复性工作自动化、模板化。
*   **可扩展性**: 一个动态模板可以服务于无限数量的设备，证明了您设计的系统具备良好的可扩展性。
*   **降低风险**: 模板化和自动化极大地减少了因手动复制、粘贴和修改配置而引入的人为错误，提升了网络变更的准确性和安全性。

动态配置模板功能，是“链踪”项目实用性和专业性的最佳证明之一。
